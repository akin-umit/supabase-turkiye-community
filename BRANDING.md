# Branding and Trademark Policy

## Project identity

- Project name: Supabase Turkiye Community
- Repository slug: `supabase-turkiye-community`
- Description: "Turkish documentation, self-hosting improvements, and open-source community contributions for the Supabase ecosystem."

## Do not use

- Supabase logos, icons, wordmark, or mascot without explicit permission
- "Official", "Supabase Cloud", or wording that implies endorsement
- Supabase's visual identity as the primary project branding

Every public README and release must retain the unconditional independent-community disclaimer and upstream links in `NOTICE.md`:

> This project is not operated, supported, sponsored, or officially endorsed by Supabase. It is an independent Turkiye community initiative that uses Supabase open-source components.

Turkish publications must use the equivalent statement:

> Bu proje Supabase tarafindan isletilmez, desteklenmez, sponsor edilmez veya resmi olarak onaylanmaz. Supabase acik kaynak bilesenlerini kullanan bagimsiz bir Turkiye toplulugu calismasidir.

This policy reduces confusion; it is not legal advice or a guarantee against a trademark claim.
