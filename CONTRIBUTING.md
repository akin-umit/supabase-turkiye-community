# Katki Rehberi / Contributing

Bu proje Turkce oncelikli bir topluluk projesidir. Katkilar herkese aciktir,
ancak yeni kullanicinin once Turkce dokumandan ilerleyebilmesi gerekir.

Human, AI-assisted, and AI-agent contributions are welcome. All contributors must read [AGENTS.md](./AGENTS.md). The human pull request author remains responsible for generated code, documentation, tests, security, and licensing.

## Turkce Oncelik

1. Sorunu ve deployment ortamini secret olmadan aciklayin.
2. Degisikligi dar kapsamli tutun ve upstream atfini koruyun.
3. Compose dogrulamasini ve ilgili smoke testlerini calistirin.
4. Migration ve rollback etkisini belgeleyin.
5. Musteriye ozel domain, kimlik bilgisi veya marka eklemeyin.
6. Etkilenen Turkce belgeleri ayni Pull Request icinde guncelleyin.
7. Ingilizce belge veya `*.en.md` dosyasi varsa Turkce belgeyle davranis olarak uyumlu tutun.
8. Is private test sunucusu veya staging ortamindan geldiyse community ozelligi olarak yayinlamadan once [COMMUNITY-PUBLICATION-FLOW.md](./COMMUNITY-PUBLICATION-FLOW.md) akisini uygulayin.
9. Dokumantasyon kurallari icin [DOCUMENTATION-MAINTENANCE.md](./DOCUMENTATION-MAINTENANCE.md) sozlesmesini uygulayin.

## English Summary

Contributions are welcome. This repository is Turkish-first: update the Turkish
guide first, keep English mirrors consistent when they exist, never publish
secrets or private deployment evidence, and follow
[DOCUMENTATION-MAINTENANCE.md](./DOCUMENTATION-MAINTENANCE.md) plus
[COMMUNITY-PUBLICATION-FLOW.md](./COMMUNITY-PUBLICATION-FLOW.md).
