# Branding and Trademark Policy

## Project identity

- Project name: OpenBase Stack
- Repository slug: `openbase-stack`
- Description: "Community self-hosting distribution based on the open-source Supabase Docker stack."

## Do not use

- Supabase logos, icons, wordmark, or a confusingly similar mascot as this project's identity
- "Official", "Supabase Cloud", or wording that implies endorsement
- Supabase's visual identity as the primary project branding

Every public README and release must retain the independent-project disclaimer and upstream links in `NOTICE.md`.

This policy reduces confusion; it is not legal advice or a guarantee against a trademark claim.
