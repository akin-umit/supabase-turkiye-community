# Branding and Trademark Policy

## Project identity

- Project name: Supabase Turkiye Community
- Repository slug: `supabase-turkiye-community`
- Description: "Turkish documentation, self-hosting improvements, and open-source community contributions for the Supabase ecosystem."

## Do not use

- Supabase logos, icons, wordmark, or mascot without explicit permission
- "Official", "Supabase Cloud", or wording that implies endorsement
- Supabase's visual identity as the primary project branding

Every public README and release must retain the independent-community disclaimer and upstream links in `NOTICE.md`. If Supabase formally recognizes the community, the wording may be updated with written evidence.

This policy reduces confusion; it is not legal advice or a guarantee against a trademark claim.
